Primeiros computadores

1ª geração:Mark I - 1944, colossus - 1946, eniac - 1946.
Curiosidade: em 1946 surgiu o termo “bug”, por causa do tamanho dos computadores e de sua temperatura, atraiam muitos insetos que acabavam prejudicam o funcionamento da máquina.
2ª geração: 1959 a 1965: evolução das válvulas para os transistores - começa a diminuir os tamanhos dos computadores.
3ª geração: 1965 a 1970: circuito integrado: transistor, capacitores e resistores.
4ª geração: a partir de 1971: começou a surgir os computadores pessoais.

O que são clients

Voltar

Cliente no linguajar web

Clientes somos nós, as pessoas que acessam o site através do navegador, essa informação chega ao servidor e retorna com a apresentação do site.

Navegadores

  • São programas criados por empresas, utilizados para abrir/executar arquivos que vem do servidor
  • Seguem padrões W3C, porém, sempre uma diferença ou outra de interpretação
  • Também chamados de browsers
  • Iniciou com o MOSAIC, passou para o netscape e hoje temos uma variedade de navegadores disponíveis
  • São gratuitos
  • Pesquisar browse support HTML da w3school, lá tem as compatibilidades dos navegadores com as tags

Sobre servidores

Softwares:

  • Sistemas operacionais
  • Monitoramento
  • Servidores web

Hardwares e softwares em servidores

  • Armazenamento (discos rígidos, SSDs)
  • Memória
  • Processamento

Tipos de servidores

  • Servidor proxy: bloqueia ou não certos tipo de sites ou informações
  • Servidor firewall: usado para segurança
  • Servidor Web server: servidor para sites
  • Database server: guarda informações mais significativas
  • Email server: servidor para emails
  • Servidor DNS (domain name service): redireciona o ip para um endereço
